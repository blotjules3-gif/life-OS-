Questions de configuration → Notifications

Chaque notification a sa propre question d'heure. Rien n'est calculé automatiquement.

Principe : 2 types de questions séparées pour chaque module.
Questions événement (gris) → à quelle heure se passe l'événement — utilisé par le coach IA pour le contexte.
Questions notification (bleu) → à quelle heure tu veux LA notification — utilisé par ContextualNotifications pour planifier.
Si la question notification n'a pas été posée → clé absente → aucune notification envoyée.
